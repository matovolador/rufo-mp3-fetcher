# RUFO MP3 FETCHER

## About

This software is a python wrapper for ffmpeg and youtube_dl. It uses tkinter UI.

## Requirements

You will need [ffmpeg](https://ffmpeg.org/) in your system PATH.
Python 3.6.x
Virtualenv

## Installation

```bash
sudo ./install-extras.sh
```

```bash
sudo ./install.sh
```

## Execution

```bash
./run.sh
```
